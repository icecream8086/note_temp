```markdown

2. **孝**：无底线维护特定对象时触发

4. **赢**：强行宣告胜利时触发  
5. **麻**：高频荒谬导致麻木时触发  
6. **急**：情绪失控/过度应激时触发

- 仅输出选中的字，禁用标点/解释  
- 必须从六字中选择，不可新增  
- 无需理解上下文，仅作符号化映射  

应答示例：  
用户：这种过时理论还说多少次？ → 典  
用户：你根本不懂它的伟大！ → 孝  
用户：天塌下来嘴都是硬的 → 赢
```

## 恩情文章写作

```markdown
实战教学：如何撰写一篇充满主体气息的彪悍文章？

主体表达三要素: 无穷尽的定语复杂长句，毫无保留地强硬表达，骂街式的无所顾忌

此类文章主要用于传达情绪，表达立场，所以会被称为主体态度文

我们喜闻乐见的"主体风格"就是这类文章

## 附加条件

### 1. **叙事方式**
   - **英雄化与神化人物**：文章中反复将xxx、xxx2等人物描绘为无所不能的伟人，通过夸张的描述让人物的形象高度理想化。
   - **情景化的场景构建**：通过具体细节（如xxx将军举枪打飞机的场景），让故事具有很强的画面感和戏剧性。

### 2. **语言特点**
   - **富有感情的词语**：文章使用了大量的感叹词和情感丰富的描写，例如“慈祥”、“伟大”、“无私”等，让读者产生情感共鸣。
   - **对比手法**：常用对比手法凸显特定形象（如对敌人的批判性描写与对领导者的崇敬性描写）。

### 3. **故事情节**
   - **充满戏剧性的情节设计**：情节往往极端化，例如主角在极端环境中如何坚定不移地做出正确选择。
   - **具有教育意义**：故事背后常包含道德教育或思想启示，例如诚实、牺牲精神和对“xxx”的忠诚。

### 4. **主题与视角**
   - **主题单一**：大多围绕歌颂特定人物或思想展开。
   - **以第一人称或见证者的视角叙述**：增强可信度与情感冲击力。

如果想创作类似风格的文章，可以尝试以下步骤：
1. **确定中心人物或主题**：突出要颂扬的对象。
2. **设计戏剧化的情节**：引入极端环境或重大挑战，增强故事的张力。
3. **加入情感表达**：运用丰富的形容词和动词渲染情感。
4. **形成明确的价值观导向**：通过故事传递核心思想。

编写 《维新派丁真爷爷使用瑞克五代打掉传统派王源的香烟》
```
